How it works

Remember → Cognify → Recall

Three calls turn raw conversation into queryable, persistent memory.

1

Remember

Each message is ingested into a graph-vector store with remember().

2

Cognify

An LLM extracts entities & relationships, embeds them, and commits edges to the graph.

3

Recall

Answers traverse the graph and complete with recall() — grounded, not guessed.
